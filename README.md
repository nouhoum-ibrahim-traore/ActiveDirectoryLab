<h1>Active Directory Home Lab</h1>

 ### [YouTube Demonstration](https://www.youtube.com/watch?v=MHsI8hJmggI&list=PL1u-esq4ni6UyoTq29gMLsYQpTO8I6LUi&index=9)

<h2>Description</h2>
In this lab we learn how to set up a basic home lab environment using Oracle VirtualBox to run Active Directory (AD). This lab covers creating and configuring a virtual machine, installing AD, and managing users efficiently with PowerShell. Perfect for beginners looking to build hands-on experience in Active Directory administration and PowerShell scripting!
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Diskpart</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
